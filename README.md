# Resumen de el PDF:
*   **Concepto:** Son como un **inventario o mochila** que agrupa diversos tipos de datos en una sola estructura organizada.
*   **Índices:** Cada elemento tiene una posición que **comienza en 0** y termina en **-1 para el último** objeto.
*   **Acciones clave:** Se usa `.append()` para agregar, `.pop()` para eliminar, y `len()` para conocer el tamaño de la lista.
*   **Superpoder:** Las **List Comprehensions** permiten filtrar y transformar grandes cantidades de datos de forma rápida y elegante.
*   **Meta:** Dominar estas herramientas para realizar tareas de **análisis de datos reales**, como calcular promedios y filtrar preferencias específicas.
